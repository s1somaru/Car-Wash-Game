# パーフェクト洗車チャレンジ

**パーフェクト洗車チャレンジ**は、洗車機をドラッグ操作で動かしながら、「画面に降ってくる汚れをすべて落としつつ、目標時間ピッタリに洗車を完了させる」ことを目指す、新感覚のマルチタスク＆体内時計ゲームです。

## 🎮 遊び方 (How to Play)

1. 画面に表示される **「目標時間」** をしっかりと確認します。
2. **[ 洗車スタート ]** ボタンを押すと洗車作業とタイマーが開始します。
3. 洗車機（青と緑の回転ブラシの柱）をマウスまたはタッチで **左右にドラッグ** し、降ってくる汚れに重ねてゴシゴシ落としていきます。
4. 開始から3秒で「経過時間」がぼやけて見えなくなるため、頭の中で秒数をカウントし続けます。
5. 目標時間が来た！と思うタイミングで **[ 洗車ストップ ]** ボタンを押します。
6. 誤差と汚れの有無が評価され、結果が表示されます！
